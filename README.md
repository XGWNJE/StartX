## 📦 安装与使用

1.  **下载代码**:
    -   点击此页面右上角的 `Code` 按钮，然后选择 `Download ZIP`。
    -   下载后，将 `StartX.zip` 解压到你喜欢的位置。

2.  **加载扩展**:
    -   打开 Chrome 浏览器，在地址栏输入 `chrome://extensions` 并回车。
    -   在打开的扩展程序页面，确保右上角的 **"开发者模式"** 开关是打开的。
    -   点击左上角的 **"加载已解压的扩展程序"** 按钮。
    -   在弹出的文件选择窗口中，找到并选择你刚刚解压的 `StartX` 文件夹。

3.  **完成**:
    -   `StartX` 现在应该出现在你的扩展列表中了。
    -   打开一个新的标签页，开始体验吧！

## ✨ 主要功能

- **搜索框**: 快速搜索网页或直接打开你的书签，所有操作一气呵成。

- **书签搜索**: 使用 `/` 前缀快速搜索您的浏览器书签，无需打开书签管理器。

- **个性化设置**:
  - **自定义壁纸**: 上传你最喜欢的图片作为背景，或在内置壁纸中选择。
  - **切换搜索引擎**: 在 Google、Bing、Baidu 之间一键切换。
  - **玻璃质感**: 随时开启或关闭优雅的毛玻璃透明效果。
  - **多语言支持**: 界面支持中文、英文、日文和韩文四种语言，自动检测浏览器语言或手动切换。

- **键盘操作**:
  - 使用 `↑` / `↓` / `Tab` 选择建议项。
  - 按 `Enter` 键即可打开链接或执行搜索。
  - `Esc` 键可用于取消选择。

- **视觉体验**:
  - **标志与品牌**: 中央显示的精美Logo和艺术字，搜索时自动淡出。
  - **模块化CSS**: 代码结构清晰，易于维护和扩展。

## 命令功能

### 书签检索（/ 前缀）

- **使用方式**：在搜索框中输入斜杠`/`作为前缀，后跟要搜索的书签关键词
- **功能**：快速搜索和访问已保存的浏览器书签
- **显示内容**：匹配的书签列表，包括书签标题和URL信息
- **操作方式**：
  - 键盘方向键上下移动选择书签
  - Enter键直接打开选中的书签
  - 鼠标点击书签项也可打开

**UI特点**：
- 简洁列表式布局，清晰显示书签信息
- 书签项有悬停高亮效果
- 与整体界面风格一致的毛玻璃效果

**示例**：
- `/` → 显示所有书签
- `/github` → 显示包含"github"的书签
- `/news` → 显示包含"news"的书签
